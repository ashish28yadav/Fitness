## 🚀 Overview

A premium, highly-animated fitness tracking dashboard built with **Jetpack Compose**. This project demonstrates modern Android development practices, custom UI components, and fluid user experiences inspired by top-tier fitness applications.

---

## ✨ Features

- **Activity Rings**: Apple-style Move, Exercise, and Stand rings with smooth stroke animations.  
- **Glassmorphic Charts**: Custom-drawn step charts with "liquid filling" effects, wave animations, and specular highlights.  
- **Interactive Hydration**: A water tracking system with spring-physics-based progress bars.  
- **Real-time Animations**: Pulse effects for heart rate, smooth transitions between summary and detail views, and staggered bar chart entries.  
- **Immersive UI**: Full edge-to-edge design with a blended notification bar and a deep vibrant gradient theme.  
- **Architecture**: Built using MVVM pattern, Hilt for Dependency Injection, and state-driven UI logic.  

---

## 🛠 Tech Stack

- **UI**: Jetpack Compose (100%)  
- **Navigation**: Compose Navigation with AnimatedContent  
- **DI**: Dagger Hilt  
- **Architecture**: MVVM  
- **Animations**: Compose Animation (Animatable, InfiniteTransition, Spring Physics)  
- **Drawing**: Compose Canvas for custom rings and liquid graphs  

---

## 🚀 Getting Started

1. Clone the repository  
2. Open in **Android Studio Jellyfish** or newer  
3. Build and run on emulator or physical device (API 26+)
